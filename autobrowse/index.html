Autobrowse jQuery plugin

Download and more documentation: jquery.esn.autobrowse.js. LICENCE: Do whatever you want with the code. We don't take any responsibility for how you use it.

This plugin adds automatic ajax loading of page content when the user scrolls the page. It comes with a browser cache, which solves the problem of a user clicking away from the page, and then goes back and ends up in the beginning of the page, instead of where she was looking. Simply use autobrowse(options) on a container that you want to fill with content.

Example usage:


$(".page").autobrowse(
  urlBuilder: function (offset, count)
  {
    return "http://api.flickr.com/services/feeds/photos_public.gne?tags=cat&tagmode=any&format=json&jsoncallback=?";
  },
  template: function (response)
  {
    var markup='';
    for (var i=0; i<response.items.length; i++)
    {
      markup+='<a href="'+response.items[i].link+'"><img src="'+response.items[i].media.m+'" /></a>'
    }
    return markup;
  },
  itemsReturned: function (response) { return response.items.length; },
  count: 20,
  offset: 0,
  totalCount: 100,
  loader: '<div class="loader"></div>',
  useCache: true
);

Note that the demo urlBuilder does not use offset and count in the url. See the source code for more examples and option details.
